السلام عليكم يا ابطال

انتوا اخترتوا المتوسط معناته خلفيتكم هي:
- العناوين
- الفقرات
- التعليقات
- الروابط
- الصور
- div
- span
- جداول
- قوائم
- دمج صفحات html
- نماذج الادخال

افكار للموقع:
- طريقه مشاهده فيديوهات المسار
- بناء اي موقع يخطؤ على البال

متطلباتكم هيا التالي:
- استعمال رسوم العناوين بناء على الصوره<h1> <h2> .... <h6> 
- استعمال رسم الفقره <p> 
- اضف تعليقات بين الوسوم حتى تبين بداية و نهاية الوسوم
- ثلاث صفحات 
- قائمه تحمل عناوين الصفحات لتسمح للمستخدمين بالتنقل بينهم
- جدول للصور
- صور تنقلنا لمواقع خارجيه

تذكير :
- لا تنسون استخدام html:5
- وضع عنوان للموقع

تحذيرات :
- انتبه الى بدايه و نهاية الوسوم
- انتبه على ان يتشابه وسم البدايه و النهايه
- انتبه على الكتابه داخل وسم <body>

بونص:
- صفحه لنماذج و حقول الادخال 
(فقط شكلا - ليس بالضروره تفعيلها)
- استعمال النوعين من القوائم
